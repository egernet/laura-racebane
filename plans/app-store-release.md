# App Store Release — Racebane

Komplet guide til at få Racebane på App Store: screenshots, beskrivelse,
metadata, privatlivspolitik og test-flow.

---

## 1. Pris og målgruppe

- **Pris**: Tier 1 (8 kr DK / $0.99 US)
- **Aldersmærke**: 4+
- **Kategori**: Spil → Racerløb (primær), Spil → Familie (sekundær)
- **Enheder**: iPhone og iPad (universal)

### Krav

App Store kræver screenshots i specifikke opløsninger per device-kategori.
Racebane er universal (iPhone + iPad), så begge skal uploades.

| Device | Opløsning (landscape) | Krævet? |

### Marketing-overlay-tekster

Lægges oven på hvert skud som stor headline:

1. "Bare gas — men slip i svinget!"
2. "Fire baner, tre sværhedsgrader"
3. "Op til 4 biler på sporet"
4. "Sæt banen på dit bord — i AR"
5. "Klar, parat — GAS!"
6. "Hvem vinder pokalen?"

Typografi: hvid sans-serif, mindst 80pt på 2752px-bredde, blød skygge.
Værktøjer: Canva (gratis App Store template), Figma, eller Picsew.

### Sådan fanger du dem

```bash
./scripts/capture-screenshots.sh
```

Scriptet:
- Tjekker at en Simulator er booted og er en 13" iPad eller iPhone Pro Max
- Walker dig gennem de 6 scener én ad gangen
- Gemmer som `screenshots/0X-name.png` (mappen er gitignored)
- Advarer hvis billedet kommer ud portrait (rotér Simulator med ⌘+→)

**Forberedelse i Simulator (iPad):**
1. Åbn Xcode → vælg destination **iPad Pro 13-inch (M4)** → kør Racebane
2. I Simulator-vinduet: ⌘+→ for at sikre landscape
3. Kør scriptet og følg instruktionerne
4. Flyt PNG'erne til `screenshots/ipad/` bagefter

**Forberedelse i Simulator (iPhone):**
1. Kør destination **iPhone 15 Pro Max**
2. ⌘+→ for landscape
3. Kør scriptet igen
4. Flyt PNG'erne til `screenshots/iphone/`

## 4. Privatlivspolitik

App Store Connect kræver en URL der hoster en privatlivspolitik.

### Tekst (kopier til Gist eller GitHub Pages)

```markdown
# Privacy Policy — Racebane

Last updated: 18 May 2026

Racebane is a slot car racing game for iPhone and iPad. We respect your
privacy.

## What we collect

**Nothing.** Racebane does not collect, transmit, or store any personal
information about you or your device.

## Camera access (AR mode)

Racebane's optional AR mode uses the device camera to display the track
on a real surface. The camera feed is processed entirely on-device and
is never recorded, saved, or transmitted anywhere. As soon as you leave
AR mode, the feed stops.

## Local network access (multiplayer)

Racebane's optional multiplayer uses Apple's MultipeerConnectivity
framework to discover and connect to other Racebane players on the same
WiFi network. No data leaves the local network, no servers are involved,
and no accounts are required.

## Internet access

Racebane does not require an internet connection. The app does not
communicate with any server, advertising network, or analytics service.

## Children's privacy

Racebane is designed for children and complies with COPPA. Because the
app collects no data, there is nothing to disclose, share, or delete.

## Contact

If you have questions about this policy, contact:
christian.enevoldsen@framna.com
```

## 5. App Store Connect checkliste

Klik gennem App Store Connect → din app i denne rækkefølge:

### App Information

- [ ] **Subtitle**: "Klassisk racerbane i lommen"
- [ ] **Primary Category**: Games → Racing
- [ ] **Secondary Category**: Games → Family
- [ ] **Content Rights**: "Does your app contain, show, or access any
      third-party content?" → **No**
- [ ] **Age Rating** → Edit → alle spørgsmål **"None"** → resultat 4+

### App Privacy

- [ ] **Privacy Policy URL**: din Gist/Pages URL
- [ ] **Data Collection**: "We do not collect data from this app"

### Pricing and Availability

- [ ] **Price**: Tier 1 (8 kr / $0.99)
- [ ] **Availability**: alle lande (eller vælg specifikke)

### Version 1.0

- [ ] **Promotional Text**: kopieret fra ovenstående
- [ ] **Description**: kopieret fra ovenstående (dansk + engelsk)
- [ ] **Keywords**: kopieret fra ovenstående
- [ ] **Support URL**: kan være samme som privacy URL eller email-link
- [ ] **Marketing URL**: valgfri
- [ ] **Screenshots** (iPad 13"): upload de 6 PNG'er
- [ ] **Screenshots** (iPhone 6.7"): upload de 6 PNG'er
- [ ] **App Review Information**:
      - Sign-in required: **No**
      - Contact: dit navn, telefon, email
      - Notes: "Single-player and local-multiplayer racing game for
        children. No accounts, no online features, no data collection.
        Multiplayer uses Apple's MultipeerConnectivity (local WiFi
        only). AR mode is optional and processes camera entirely
        on-device."

### Build

- [ ] Archive i Xcode → upload til App Store Connect
- [ ] Vent ~10 min på processing
- [ ] Vælg den nye build i version 1.0
- [ ] Eksportér compliance: **No** (ingen kryptering)

## 6. Test før upload

Kør i Simulator + på fysisk iPad/iPhone inden archive:

### Simulator (iPad Pro 13" + iPhone 15 Pro Max)
- [ ] Spillet starter i landscape (på iPhone) og virker i begge orienteringer (iPad)
- [ ] Menu vises og banevælger viser alle 4 baner
- [ ] Race på alle 4 baner virker uden crash
- [ ] Gas-knappen reagerer øjeblikkeligt
- [ ] Afkørsel: bil flyver af i kurver ved for høj fart, 2 sek straf
- [ ] AI på alle 3 sværhedsgrader færdiggør banen
- [ ] HUD viser fart, omgange og position korrekt
- [ ] Countdown 3-2-1-GO vises før hvert race
- [ ] Resultatskærm vises efter mål med korrekt vinder
- [ ] Luk-knap virker og fører tilbage til menu
- [ ] App krasher ikke ved app-switch (background/foreground)
- [ ] AR mode er disabled i Simulator (ARKit kræver fysisk device)

### Fysisk iPad/iPhone
- [ ] Alt fra Simulator-listen virker
- [ ] AR mode: planedetection finder bordet/gulvet
- [ ] AR mode: bane spawner og bilen kører rundt
- [ ] Multiplayer: host + 2-3 clients kan finde hinanden over WiFi
- [ ] Multiplayer: synkroniseret start, ingen sync-fejl over 30 sek race
- [ ] Multiplayer: restart efter race virker
- [ ] Haptic feedback ved afkørsel
- [ ] Lyd virker (motor, eksplosion, countdown, menu-musik)
- [ ] Performance er stabil 30+ fps med 4 biler

### Build-konfiguration
- [ ] `project.yml`: `TARGETED_DEVICE_FAMILY: "1,2"` (universal)
- [ ] `project.yml`: landscape + portrait på iPad, landscape-only på iPhone
      (overvej at låse iPhone til landscape — race-spil)
- [ ] `INFOPLIST_KEY_NSCameraUsageDescription` sat (AR-mode)
- [ ] `NSLocalNetworkUsageDescription` sat (multiplayer)
- [ ] `NSBonjourServices: [_racebane._tcp]` sat
- [ ] App ikon vises på home screen (alle størrelser)
- [ ] Launch screen kører kortvarigt uden visuelle fejl
- [ ] `MARKETING_VERSION: "1.0"` i project.yml

## Nyttige kommandoer

```bash
# Regenerér Xcode projekt efter project.yml ændring
xcodegen generate

# Build for Simulator
xcodebuild -project Racebane.xcodeproj -scheme Racebane \
  -destination 'platform=iOS Simulator,name=iPad Pro 13-inch (M4)' \
  -configuration Debug build

# Build for fysisk iPad
xcodebuild -project Racebane.xcodeproj -scheme Racebane \
  -destination 'platform=iOS,name=Lauras iPad' \
  -configuration Debug build

# Fang screenshots
./scripts/capture-screenshots.sh

# Tjek opløsning på et screenshot
sips -g pixelWidth -g pixelHeight screenshots/01-race.png
```
